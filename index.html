📋 Cómo configurar Google Sheets
🗑 Limpiar datos

Borra todos los pedidos y gastos guardados en el celular (los que ya se enviaron a Google Sheets no se borran allá).

☁️ Configurar Google Sheets
1
Ve a drive.google.com desde el celular o PC → crea una hoja nueva → nómbrala "FB Ventas".
2
En la hoja: clic en Extensiones → Apps Script.
3
Borra todo el código que aparece y pega exactamente este código:
function doPost(e) { try { var ss = SpreadsheetApp.getActiveSpreadsheet(); var data = JSON.parse(e.postData.contents); if (data.tipo === "pedidos") { var sh = ss.getSheetByName("VENTAS"); if (!sh) { sh = ss.insertSheet("VENTAS"); sh.appendRow(["CLIENTE","TELEFONO","BARRIO", "DIRECCION","PRODUCTO","CANTIDAD","PRECIO", "TOTAL","ABONO","FORMA_PAGO","ESTADO", "FECHA","HORA","OBSERVACIONES","SINCRONIZADO"]); sh.getRange(1,1,1,15).setBackground("#1A2B6D") .setFontColor("white").setFontWeight("bold"); } data.filas.forEach(function(f){ sh.appendRow(f); }); } if (data.tipo === "gastos") { var shG = ss.getSheetByName("GASTOS"); if (!shG) { shG = ss.insertSheet("GASTOS"); shG.appendRow(["TIPO","DESCRIPCION","MONTO", "PROVEEDOR","FECHA","HORA"]); shG.getRange(1,1,1,6).setBackground("#EA580C") .setFontColor("white").setFontWeight("bold"); } data.filas.forEach(function(f){ shG.appendRow(f); }); } if (data.tipo === "ping") { return ContentService .createTextOutput(JSON.stringify({ok:true,msg:"Conexión OK"})) .setMimeType(ContentService.MimeType.JSON); } return ContentService .createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({ok:false,error:err.message})) .setMimeType(ContentService.MimeType.JSON); } }
4
Clic en Implementar → Nueva implementación → tipo: Aplicación web → Ejecutar como: Yo → Quién tiene acceso: Cualquier persona → clic Implementar.
5
Copia la URL que aparece (empieza con https://script.google.com/macros/s/...).
6
Vuelve a esta app → pestaña Config → pega la URL → toca Guardar configuración.
7
Toca Probar conexión para verificar que todo funciona. Si dice Conexión OK ya está listo. 🎉
